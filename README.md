- 1. highway_yolov8_data_cleaning.ipynb : 고속도로 CCTV 이미지 및 라벨 파일 전처리

- 3. test_video.avi : 생성한 yolov8m 모델로 실시간 고속도로 CCTV 객체 탐지 실습 동영상 (약 1분 30초)
